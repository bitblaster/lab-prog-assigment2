La riga di comando usata per compilare è:
	g++ *.cpp parsers/*.cpp -o lab-prog-assigment2 -O2

Il programma può essere lanciato con alla fine l'argomento -v per abilitare un output più verboso (ad es. stampa gli ordini non evasi)

./lab-prog-assigment2 file-ordini.dat

nella dir corrente sono presenti 4 file output*.txt con l'output del programma lanciato con i relativi file orders*.dat
